# Ford Triplog

Automatic trip logging for Ford vehicles using the FordPass integration.

## Features

- Automatic trip detection
- Smart Trip (merge short stops)
- Config Flow & Options Flow
- Start/End addresses
- GPS coordinates
- Odometer & SOC
- Home Assistant 2026.6+
- Brands Proxy API support

## Installation

1. Copy `custom_components/ford_triplog` into your Home Assistant `custom_components` directory.
2. Restart Home Assistant.
3. Add **Ford Triplog** via **Settings → Devices & Services**.
4. Select the required entities (Ignition, Odometer, Device Tracker, optional SOC).
